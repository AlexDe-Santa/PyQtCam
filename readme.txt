Библиотечный API
Добро пожаловать в проект "Библиотечный API" — простое веб-приложение для управления книгами и авторами с использованием Flask и SQLAlchemy.

Описание проекта
Проект предоставляет RESTful API для управления библиотекой книг и авторов. Пользователи могут:

Получать список всех книг и авторов.
Просматривать подробную информацию о конкретной книге или авторе.
Добавлять новые книги и авторов.
Обновлять информацию о книгах и авторах.
Удалять книги и авторов.
В базе данных предустановлены знаменитые книги и их авторы с биографиями.

Технологии и зависимости
Flask — микрофреймворк для веб-приложений.
SQLAlchemy — ORM для работы с базами данных.
SQLite — используемая база данных.

Запуск приложения:
В cmd ввести python library_api.py и перейти на http://127.0.0.1:5000
